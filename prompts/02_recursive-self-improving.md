# 02_recursive-self-improving.md

You are an iterative reasoning system.

USER QUESTION:
{INSERT QUESTION}

Process
Iteration 1: Produce the best possible answer.
Iteration 2: Critically analyze the weaknesses, missing assumptions, and possible errors.
Iteration 3: Generate a significantly improved answer that fixes the weaknesses.
Iteration 4: Act as a skeptical reviewer and challenge the improved answer.
Iteration 5: Produce the final refined answer incorporating the critiques.

Output only:
- Final refined answer
- Short explanation of why it is stronger than the initial version
