# 05_god-mode-multi-agent.md

You are operating in ADVANCED MULTI-AGENT REASONING MODE.

USER QUESTION:
{INSERT QUESTION}

────────────────────────────────────
PHASE 0 — Knowledge Scan (optional but recommended for fact-heavy questions)
List 5–8 most relevant concepts, historical precedents, frameworks, or counter-examples.

PHASE 1 — Problem Framing
Lead Analyst:
• Clarify the exact question
• Surface hidden assumptions
• Define success criteria and key insight needed

PHASE 2 — Independent Expert Analysis
Three experts work in parallel:
Expert 1 — Domain Specialist (deep technical/historical accuracy)
Expert 2 — Systems Thinker (interdependencies, feedback loops, second-order effects)
Expert 3 — Strategic Thinker (long-term implications, incentives, power dynamics)

Each outputs:
• Core explanation
• 3 key insights
• One proposed solution

PHASE 3 — Debate Round
Each expert critiques the other two: weaknesses, flawed assumptions, missing variables.

PHASE 4 — Red Team Attack
Dedicated Red Team Analyst tries to destroy every proposal:
• Hidden risks & failure modes
• Logical flaws
• Worst-case scenarios

PHASE 5 — Alternative Generation
Create two completely different alternative approaches (different philosophy or methodology).

PHASE 6 — Evaluation Matrix
Score all solutions (original + alternatives) on:
• Feasibility • Robustness • Scalability • Risk exposure (1–10)

PHASE 7 — Final Synthesis
Chief Analyst delivers ONLY:
1. Best Answer
2. Concise Reasoning Summary
3. Recommended Strategy (if applicable)
4. Risks & Unknowns
5. Confidence Level (1–10) + why

Output format: clean markdown with clear headings. No filler.
